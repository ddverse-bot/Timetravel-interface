Time-Travel UI

A futuristic, scroll-driven web experience showcasing UI evolution from 2005 → 2025 → 2035.

This project is a frontend-only demo combining GSAP scroll animations, WebGL visualizations with Three.js, and modern UI/UX concepts including a frosted-glass future interface.

Features
2005 - The Past

Comic Sans fonts, heavy buttons, classic popups.

Simulates old-school UX.

Scroll-triggered fade-in animations.

2025 - The Present

Clean, minimal, responsive UI.

Modern typography (Inter font).

Smooth scroll animations and subtle text morphing.

2035 - The Future

Futuristic Orbitron font.

WebGL wireframe icosahedron spinning in background.

Frosted-glass effect on the interface section.

Gesture zone scales based on cursor proximity.

Voice hint placeholder.


Audio

Era-specific audio plays when section is active.

Audio unlocks on first click to comply with browser autoplay policies.

Technical Details

GSAP + ScrollTrigger for scroll-based animations.

Three.js for 3D WebGL objects.

Responsive design, works on modern browsers.

All features are frontend-only.
